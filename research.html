Research Projects

  1. SwapAdvisor: Support Large Deep Learning Models via Static Planning Tensor Swap. [2018 - present]
    2. SwapAdvisor helps users to explore large deep learning models by statically analyzng the dataflow graph and memory constraints to provide a good tensor swapping plan that incurs minimized swapping overhead (maximized the overlap between communication and compuation).
  2. Tofu: Distributing Tensor Computation Automatically for Large-scale Deep Learning. [2016 - 2018]
    3. Tofu is a system that partitions very large deep neural network models across multiple GPU devices to reduce per-GPU memory footprint. Tofu achieves good results through automatically choosing best partition strategies among data parallelism, model parallelism and hybrid parallelism.
  3. Spartan: Distributed Array Programming Framework. [2014 - 2015] GitHub
    4. Spartan is a distributed array framework which provides several data-flow high-level operators to help users to implement distributed array programs. The 'key' for these operators is 'extent' which is a data structure representing the shape and location of the 'value', a 'tile' (sub-array). With this design, Spartan can automatically partition the arrays across hundreds of machines while reducing communication overhead.
